# Simple HTML Project

This is a **basic HTML project** created as part of the **GDG (Google Developer Group)** learning activity.  
It demonstrates fundamental web development concepts using only **HTML**.

##  Features
- Clean and simple webpage structure  
- Basic HTML tags and layout  
- Easy to understand and edit  
- Beginner-friendly project to practice web fundamentals  

##  How to Use
1. Clone or download this repository.  
2. Open the project folder.  
3. Double-click on `index.html` to open it in your browser.
